## Overview

A rigorous, reproducible empirical academic paper and Python codebase investigating the persistent valuation discount of South Korean equities ("Korea Discount"). The milestone delivered a 20-year panel dataset, full econometric analysis suite (stacked event study, panel OLS, synthetic control, robustness checks), and a 48-page submission-ready LaTeX PDF with a two-command replication package.

## Milestone Summary

**Key Accomplishments:**
1. Reproducible Python repo with locked event dates (config.py), pinned deps, and canonical `panel.parquet` (20-year monthly KOSPI/TOPIX/S&P500/MSCI EM panel)
2. Korea Discount quantified: -0.177x P/B vs TOPIX, -0.601x vs MSCI EM (HAC-robust, Newey-West)
3. Stacked event study (Cengiz 2019 design) + panel OLS with wild-bootstrap p-values + GPR geopolitical sub-analysis — Figures 2 & 3, Tables 2 & 3
4. Synthetic control (pysyncon ADH 2010), placebo falsification (Taiwan, Indonesia), P/E replication, alternative EM control group robustness suite
5. 48-page submission-ready LaTeX PDF (`paper/main.pdf`, 370 KB) compiled clean via `latexmk`; all figures and tables included programmatically
6. Full replication package: `run_all.py` orchestrates 11 scripts in dependency order; two-command README workflow

**Key Decisions:**
- Python over R (user preference)
- Staggered event study + panel OLS as primary identification methods; synthetic control as robustness
- All three Japan reform dates as treatment events (2014, 2015, 2023) — stacked cohort gives three bites
- `linearmodels.PanelOLS` for two-way fixed effects (not statsmodels)
- Event-date firewall in `config.py` before any data loading (look-ahead bias prevention)
- `pysyncon==1.5.2` ADH implementation for synthetic control
- High synthetic-control RMSPE (0.2893) accepted as paper-text caveat rather than blocking defect
- Korea Discount units in P/B multiple (ratio) not basis points — direct interpretation

**Technical Debt / Known Issues:**
- Synthetic-control RMSPE of 0.2893 is high; framed as caveat in Discussion/Limitations section
- N=4 countries limits formal pre-trends testing power; synthetic control pre-fit used as substitute
