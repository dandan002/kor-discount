| Boundary | Description |
|----------|-------------|
| panel input -> robustness runner | The runner trusts the canonical panel schema and must fail clearly if required columns or windows are unavailable. |
| policy catalog -> artifact paths | Spec names control where robustness artifacts are written; incorrect mapping can overwrite or mislabel outputs. |
| tests -> generated artifacts | The Phase 8 gate must execute fresh generation to avoid stale artifacts in `output/` hiding a broken runner. |
