Implement `build_stacked_dataset(panel: pd.DataFrame) -> pd.DataFrame`:

1. Read only the supplied `panel` dataframe; do not read any other analysis output.
2. Restrict to `date <= STUDY_END`.
3. Pivot P/B values to date x country, compute `spread = KOSPI - TOPIX`.
4. For each `event_date` in `config.EVENT_DATES`, compute `event_rel_time` in whole calendar months.
5. Keep rows with `STACK_WINDOW_MIN <= event_rel_time <= STACK_WINDOW_MAX`.
6. Exclude rows that fall within `STACK_WINDOW_MIN <= other_rel_time <= STACK_WINDOW_MAX` for any other event date. Add a comment containing `Cengiz et al. (2019)` explaining this clean-cohort exclusion.
7. Fit the pre-event trend on rows with `-36 <= event_rel_time <= -1` using `statsmodels.api.OLS(pre_spread, sm.add_constant(pre_event_rel_time))`.
8. Predict the counterfactual spread for all retained rows and compute `abnormal_spread = spread - expected_spread`.
9. Keep and return columns `date`, `cohort`, `event_label`, `event_rel_time`, `spread`, `expected_spread`, and `abnormal_spread`.
10. Add a code comment noting that the 2014 cohort post-period is truncated after clean-cohort exclusion because the 2015-06 reform falls inside its +24 month window.

Implement `estimate_event_study(stacked: pd.DataFrame) -> pd.DataFrame`:

1. Restrict to `EVENT_WINDOW_MIN <= event_rel_time <= EVENT_WINDOW_MAX`.
2. Create cohort x relative-time dummy columns from `cohort` and `event_rel_time`.
3. Drop the t=-1 dummy for every cohort (`BASE_PERIOD = -1`).
4. Fit `statsmodels.api.OLS(y, X).fit().get_robustcov_results(cov_type="HC3")` where `y` is `abnormal_spread`.
5. Return a dataframe with `cohort`, `event_label`, `event_rel_time`, `coefficient`, `hc3_se`, `t_stat`, `p_value`, and `car`, where `car` is the cumulative sum of coefficients within each cohort ordered by `event_rel_time`.

Implement `main()`:
