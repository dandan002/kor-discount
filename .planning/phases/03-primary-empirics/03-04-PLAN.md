GPR_ESCALATION_QUANTILE = 0.75

Implement `load_gpr_korea() -> tuple[pd.DataFrame, float]`:

1. Read `config.RAW_DIR / GPR_FILENAME` with `pd.read_excel(..., usecols=["month", GPR_COLUMN], engine="xlrd")`.
2. Rename `month` to `date`.
3. Convert `date` with `pd.to_datetime`.
4. Drop rows where `GPRC_KOR` is missing.
5. Filter to `STUDY_START <= date <= STUDY_END`.
6. Compute `GPR_KOREA_ESCALATION_THRESHOLD = study[GPR_COLUMN].quantile(GPR_ESCALATION_QUANTILE)`.
7. Create `gpr_escalation_dummy = (GPRC_KOR > GPR_KOREA_ESCALATION_THRESHOLD).astype(int)`.
8. Return the study dataframe and the threshold.

Implement `build_geo_regression_data(panel: pd.DataFrame, gpr: pd.DataFrame) -> pd.DataFrame`:

1. Restrict panel to `date <= STUDY_END`.
2. Pivot P/B values to date x country.
3. Create `kospi_pb = KOSPI`, `topix_pb = TOPIX`.
4. Merge with the GPR dataframe on month-end dates. If GPR dates are first-of-month, convert both panel and GPR dates to monthly periods and merge on that period.
5. Add `year = date.dt.year.astype("category")`.
6. Return rows with non-null `kospi_pb`, `topix_pb`, and `gpr_escalation_dummy`.

Implement `estimate_geo_model(reg_df: pd.DataFrame)`:

```python
model = smf.ols("kospi_pb ~ gpr_escalation_dummy + topix_pb + C(year)", data=reg_df).fit()
hc3 = model.get_robustcov_results(cov_type="HC3")
```

`C(year)` is the time fixed effect for this KOSPI-only monthly regression; do not use full month fixed effects because one KOSPI observation per month would saturate the model and make `topix_pb` unidentified.

Implement `main()`:
